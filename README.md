# Quick Draw AI Sketch Generator

Generate AI sketches like Google's Quick Draw using a diffusion model.

## What You Get
- AI that draws sketches (cats, dogs, cars, etc.)
- Runs in Docker container
- Works on Windows 11
- No training needed - just run and get sketches

## Files Included
├── inference.py # Main app (generates sketches) 
├── train.py # Training script (optional) 
├── Dockerfile # Docker setup 
├── requirements.txt # Python packages 
├── quickdraw_diffusion_epoch_50.pth # AI model 
├── .dockerignore # Files to ignore 
└── README.md # This file
